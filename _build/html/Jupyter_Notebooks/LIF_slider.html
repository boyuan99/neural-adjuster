Leaky Integrate and Fire Model#

Introduction#

Compared to the non-leaky integrate-and-fire model a “leak” term in the membrane potential equation, reflecting the diffusion of ions through the membrane.

The model equation looks like:

\[C\frac{dV(t)}{dt} = I(t) - \frac{V(t)}{R}\]

where:

\(V(t)\) is the voltage across the cell membrane

\(I(t)\) is the injected current

\(C\) is the membrane capacitance

\(R\) is the membrane resistance

The membrane voltage increases until it reaches a constant threshold \(V_{th}\), thereafter the membrane potential is reset. In the perfect(non-leaky) integrate and fire neuron model, \(R\) is set to infinity.

V_init=-80

params=dict(V_T=[-50.0], V_0=[-80], V_imp=[-20],
            C=[10.0], R=[10.0], dt=[1e-4],
            )

params_source = ColumnDataSource(data=params)

def lif_step(I_ext, V, dt, params=params):
    dt = dt * 1000
    V_T = params['V_T'][0]
    V_0 = params['V_0'][0]
    V_imp = params['V_imp'][0]
    R = params['R'][0]
    C = params['C'][0]
    
    dV = (I_ext - V/R)/C
    
    V1 = V + dt * dV
    if V1 >= V_T:
        if V1 >= V_imp:
            V1 = V_0
        else:
            V1 = V_imp

    return V1

dt = 1e-4
t = np.arange(0, 1, dt)

I_ext = np.zeros_like(t)
I_ext[int(0.2*len(t)):int(0.8*len(t))] = 10

V = np.zeros_like(t)
V[0] = V_init

for i in range(len(t)):
    V_pre = lif_step(I_ext[i], V[i], dt)
    if i < len(t)-1:
        V[i+1] = V_pre

source = ColumnDataSource(data=dict(I=I_ext, V=V, t=t))

callback_V_T = CustomJS(args=dict(source=source, params_source=params_source), code="""
    let data = source.data;

    let t = data['t']
    let I = data['I']
    let dt = params_source.data['dt'][0]
    let V = Array(t.length).fill(0)

    
    V[0] = data['V'][0]

    let lif_step = function(I_ext, V, dt, params_source){
        dt = dt*1e3
        let V_T = cb_obj.value
        
        let V_0 = params_source.data['V_0'][0]
        let V_imp = params_source.data['V_imp'][0]
        let R = params_source.data['R'][0]
        let C = params_source.data['C'][0]
        
        let dV = (I_ext - V/R)/C
        
        let V1 = V + dt * dV
        if (V1 >= V_T){
            if (V1 >= V_imp){
                V1 = V_0
            }
            else{
                V1 = V_imp
            }
        }
        
        return [V1, V_T]
    }
    
    let V_T = 0;
    for (let i=0; i<t.length;i++){
        let pre = lif_step(I[i], V[i], dt, params_source)
        let V_pre = pre[0]
        V_T = pre[1]
        if (i < t.length-1){
            V[i+1] = V_pre
        }
    }
    
    params_source.data['V_T'][0] = V_T;
    source.data['V'] = V;
    
    source.change.emit();
    params_source.change.emit();
    
""")
